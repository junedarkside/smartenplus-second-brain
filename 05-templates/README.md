# 05-templates — MOC

## Summary
Note skeletons for consistent authoring. Copy, do not link.

## Contents
- [[project-readme]] — per-subfolder project README
- [[architecture-review]] — code/architecture review structure
- [[atomic-note]] — single-concept knowledge note
- [[bug-report]] — bug write-up structure
- [[research-summary]] — research output structure

## Conventions for this folder
- Templates are copy-paste, not destinations for inbound links
- Placeholder wikilinks (e.g. `[[adr-xxx]]`) inside templates are intentional
- When template changes, regenerate dependent notes (not required, recommended)
