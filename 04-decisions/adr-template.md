# ADR Template

## Summary
Architecture Decision Record template. Copy this when making a new architectural decision.

## How to Use
1. Copy this file to `04-decisions/decision-name.md`
2. Fill in each section
3. Update `index.md`
4. Add entry to `log.md`

## Template

```markdown
# ADR: [Decision Title]

## Status
[proposed | accepted | deprecated | superseded by [[adr-xxx]]]

## Context
What force or situation prompted this decision.

## Decision
What was decided.

## Alternatives Considered
1. Option A — why rejected
2. Option B — why rejected
3. Chosen option — why accepted

## Tradeoffs
- Gained: ...
- Lost: ...
- Risk: ...

## Consequences
What changes because of this decision.

## Related
- [[related-page]]
```
